Créditos y Autoría

Esta página fue creada por Francisco R. González con la ayuda de ChatGPT. En ella se utilizan elementos como texto, listas, imágenes y elementos externos incluidos mediante iframes. El propósito es aprender los principios básicos de creación de sitios web para proyectos de Humanidades Digitales.

En esta versión, el sitio fue reorganizado como un recorrido narrativo entre los colores de la bandera LGBT+. La portada invita a comenzar el trayecto y cada página funciona como un capítulo conectado con la siguiente, sin alterar el contenido central ni los materiales multimedia de cada sección.

El diseño utiliza una estructura de múltiples páginas HTML con una hoja de estilos CSS compartida. También incorpora comentarios en el código para facilitar la lectura y apoyar el aprendizaje de estudiantes que se acercan por primera vez al desarrollo web.
