Créditos y Autoría

Esta página fue creada por Francisco R. González con la ayuda de ChatGPT. En ella se utilizan elementos como texto, listas, imágenes y elementos externos incluidos mediante iframes. El propósito es aprender los principios básicos de creación de sitios web para proyectos de Humanidades Digitales.

El diseño utiliza una estructura sencilla de múltiples páginas HTML con una hoja de estilos CSS compartida, para facilitar la lectura del código y permitir que estudiantes sin experiencia avanzada en programación web puedan comprender cómo se organiza un sitio.
